# Quickstart

### Plugged to Your Search AI, Keep Your Documents Updated

**KAI (Knowledge AI)** is a platform that enhances AI-powered search systems by automatically maintaining your document repositories. We detect when your documents are outdated, contradictory, or missing critical informationâ€”then help you fix them before users experience the problem.

### What is KAI?

KAI provides intelligent document maintenance for systems using generative AI. Whether you're using RAG, agents, copilots, or any AI search system, KAI ensures your knowledge base stays accurate, consistent, and complete.

#### The Core Problem We Solve

Your AI search system is only as good as your documents. When documents are:

* **Outdated** after product changes
* **Contradictory** across different sources
* **Missing** information users are asking about

Your AI gives confusing, incorrect, or incomplete answers. Users lose trust. Support tickets spike.

**KAI solves this by automatically detecting these issues and helping you fix them.**

### Common Questions

#### Do I need both KAI Document Compagnon and KAI Instance?

**No.** KAI Document Compagnon uses KAI Instance under the hood. If you're using KAI Document Compagnon, you're already benefiting from KAI Instance's semantic technology.

**Use KAI Instance directly** if you're building your own AI search system and want semantic retrieval as a component.

#### What AI systems does KAI work with?

KAI Document Compagnon works with any AI search system that processes user queries:

* RAG systems (Retrieval-Augmented Generation)
* Conversational agents
* AI copilots and assistants
* Any system with API access to user queries

#### How does KAI connect to my systems?

**Document Repositories:** Via connectors (Confluence, SharePoint, Notion, Git, file systems, etc.)

**AI Search System:** Via REST APIâ€”your AI system sends us user queries in real-time

**No changes needed** to your existing infrastructure.

#### What document formats are supported?

* Microsoft Office: `.docx`, `.pptx`, `.xlsx`
* PDF: `.pdf`
* Markdown: `.md`
* Web: `.html`
* Text: `.txt`

_Audio and video files are not currently supported._

#### How is KAI deployed?

**KAI Instance** can be deployed as:

* **SaaS** (cloud-hosted)
* **On-premises** (your infrastructure)
* **Snowflake Marketplace** (native integration)

**KAI Document Compagnon** is typically deployed as SaaS, with on-premises options available.
